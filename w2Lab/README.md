# Week 2 Lab – Configuring a Small Network

## 📌 Overview
This lab focuses on setting up and configuring a small network using Cisco Packet Tracer. It covers basic device configuration, IP addressing, and verification of connectivity between devices.

---

## 🧩 Lab Objectives
- Set up the network topology using Ethernet connections  
- Configure PC hosts with correct IP addressing  
- Configure and verify switch settings  
- Test connectivity between devices  

---

## 🌐 Network Topology
Devices used:
- 2 PCs (PC-A, PC-B)  
- 2 Switches (S1, S2)  

---

## 🔢 IP Addressing

| Device | Interface | IP Address     | Subnet Mask       |
|--------|----------|---------------|-------------------|
| PC-A   | NIC      | 192.168.1.10  | 255.255.255.0     |
| PC-B   | NIC      | 192.168.1.11  | 255.255.255.0     |
| S1     | VLAN 1   | 192.168.1.12  | 255.255.255.0     |
| S2     | VLAN 2   | 192.168.1.13  | 255.255.255.0     |

---

## ⚙️ Key Configurations

### 🔹 PC Configuration
- Assigned static IP addresses to PC-A and PC-B  
- Verified connectivity using ping  

### 🔹 Switch Configuration
- Configured VLAN interfaces  
- Assigned IP addresses to switches  
- Verified switch communication  

---

## 🧪 Verification
- Tested connectivity between PCs  
- Verified communication with switches  
- Ensured proper network configuration  

---

## 🛠 Tools Used
- Cisco Packet Tracer (v8.2.2)  
- Basic Cisco IOS commands  

---
